# Responsible Use

This project automates a visible browser action that can contact another person. Use it carefully.

## Allowed Use

- Sending a message from an account you control
- Contacting a recipient when you have a legitimate reason and authority to do so
- Previewing a message before sending
- Testing between accounts controlled by the tester
- Auditing and improving the skill using synthetic data

## Prohibited Use

- Spam or unsolicited bulk messaging
- Harassment, intimidation, or abusive content
- Impersonation or deceptive identity claims
- Credential theft, phishing, malware delivery, or fraud
- Circumventing platform restrictions or safety controls
- Sending messages to unrelated people for testing
- Publishing private conversation content in issues or documentation

## Operator Responsibilities

- Review the recipient and message before approving a real send.
- Use Safety Rehearsal for uncertain targets.
- Respect recipient requests to stop contact.
- Follow X rules and applicable privacy, marketing, and communications laws.
- Keep task reports and screenshots protected when they contain personal data.

This repository provides software, not permission to contact any person.
